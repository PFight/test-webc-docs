Class BaseControlParams

Базовый класс для описания публичных свойств контрола.

Публичные свойства должны объявляться использованием одного из трех декораторов: r (свойство, котрое запрещено изменять после создания контрола), rw (разрешено изменять) или event (событие). Например:

@r isLoaded?: boolean;
@rw visibility?: boolean = true;
@event click?: BasicApiEvent<IClickEventArgs>;

Свойства, которые не существенны для контрола (либо имеют значения по умолчанию) должны быть помечены как необязательные (с помощью знака вопроса, например @rw compactMode?: boolean = false;). Это позволит не указывать данные свойства при создании контрола через JSX из скриптов.

Получение и запись значения публичных свойств можно переопределить в методах BaseControl.setParamValue, BaseControl.getParamValue, либо при помощи объявления свойств с декоратором handler.
