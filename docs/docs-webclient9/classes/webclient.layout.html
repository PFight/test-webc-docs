Class Layout

Protected getParamValue

  • getParamValue(paramName: string): any

  • Inherited from BaseControl.getParamValue

    • Defined in layout-extension.d.ts:2541

    Возвращает значение параметра при обращении через params объект. По умолчанию реализуется следующая логика:

    1. Если объявлено get-свойство с декоратором handler, то возвращается значение данного свойства
    2. Если свойство controlImpl содержит объект, то возвращается реультат вызова BaseControlImpl.getParamValue.
    3. Если свойство controlImpl равно undefined, то выдается предупреждение, о том что controlImpl еще не инициализирован. Данная ситуация может возникнуть, если обращение к свойству происходит до вызова componentDidMount.

    Parameters

    • paramName: string

      Имя параметра, значение которого необходимо получить

    Returns any

    Значение параметра

Protected setParamValue

  • setParamValue(paramName: string, value: any, initial: boolean): void

  • Inherited from BaseControl.setParamValue

    • Defined in layout-extension.d.ts:2565

    Обработчик, вызываемый всякий раз, когда установливается значение параметра. Происходить это может в следующих случаях:

    1. При инициализации компонента (см. setParamValues)
    2. При получении новых props компонента (ситуация возможна при создании компонента из скриптов через JSX-синтаксис, см. setParamValues).
    3. При установке значения через объект params (например params.visiblity = false).

    Метод Реализует следующую логику:

    1. Если параметр является событием (объявлен с декоратором event), то вызывается setEventValue.
    2. Если запись не разрешена (параметр только для чтения (объявлен с декоратором r) и initial = false), то сообщается об ошибке.
    3. Если запись разрешена, то а) Ищется set-свойство в текущем классе с декоратором handler для данного параметра, и если оно присутствует, то вызывается оно. б) Если свойство не найдено, значение параметра помещается в state. В методе render значения из state обычно передаются в controlImpl.

    Parameters

    • paramName: string

      Имя параметра, значение которого нужно установить.

    • value: any

      Значение параметра

    • initial: boolean

      Значение истино, если метод вызывается при инициализации компонента (из componentWillMount).

    Returns void
